# 🌐 Network Infrastructure – VLANs, Routing and ACLs

## 🎯 Overview
Projeto de implementação de rede local empresarial com segmentação por VLANs e controlo de acessos.

---

## 🛠️ Tecnologias
- VLANs
- Switch Layer 3
- Routing Inter-VLAN
- ACLs
- DHCP
- DNS

---

## 🔎 Atividades Realizadas
- Planeamento de rede e subnetting
- Criação de VLANs por departamento
- Configuração de trunking entre switches
- Implementação de routing inter-VLAN
- Configuração de ACLs para segmentação de acesso
- Testes de conectividade entre redes

---

## 🚨 Resultados
- Segmentação de rede eficiente
- Isolamento da rede Guest
- Comunicação controlada entre VLANs

---

## 📌 Notas
Projeto baseado em cenário empresarial com múltiplos departamentos.
